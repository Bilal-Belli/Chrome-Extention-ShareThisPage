# Share This Page Pro

A Chrome extension that lets you share any web page across 100+ platforms with one click.

## Features
- 100+ sharing platforms
- Search to find platforms quickly
- One-click copy to clipboard
- Clean mobile-like side panel interface

## Installation
1. Clone this repository
2. Open Chrome Extensions (chrome://extensions/)
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the extension folder

## Usage
1. Click the extension icon in your toolbar
2. Search or scroll to find your platform
3. Click the platform to copy the link
4. Paste and share!

## Platforms Supported
- Social Media: Facebook, X, Instagram, LinkedIn, etc.
- Messaging: WhatsApp, Telegram, Discord, etc.
- Cloud: Google Drive, Dropbox, OneDrive, etc.
- Developer: GitHub, GitBook, Jupyter, etc.
- And 100+ more!

## Privacy
This extension only accesses the current tab's URL and title to share it. No data is stored or transmitted to any third party.
